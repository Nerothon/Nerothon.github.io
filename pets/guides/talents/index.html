Active/Locked talents

Active talents are special talents that can be, well, activated on command. While other talents are passive and work without your intervention, active talents require you to activate them to use their effect.
There are 2 types of Active talents - Adventure talents and Combat talents.
Every active talent has a Happiness cost associated with it. Every time you use an active talent, it will cost your pet some Happiness. You can't use an active talent if the pet doesn't have enough Happiness.
Every active talent has a cooldown associated with it. Every time you use an active talent it will go on cooldown. You will have to wait for it to cooldown i norder to be able to use another active ability again.
All active talents start of locked. When you manifest them for the first time, you have to unlock them first with Tokens if you want to use them. Once you unlock it, it will stay unlocked on all YOUR offsprings of that pet. So if that pet is the parent of your next pet, the offsping will have it unlocked. This is only true for your pets/offsprings. If another player hatches with that pet, unless their pet has the talent unlocked, they will receive offspring with the talent locked, just how you originally gor it first time.

Tokens

There are 4 Ranks of tokens:
  • Rank 1 - Talent Token - generic token that is used in unlocking process of all active talents
  • Rank 2 - Triad Tokens:
    • Elemental Token
    • Spirit Token
    • Harmonious Token
  • Rank 3 - School Tokens - one for each school:
    • Balance Token
    • Death Token
    • Fire Token
    • Ice Token
    • Life Token
    • Myth Token
    • Storm Token
  • Rank 4 - Spell Tokens - these are tokens for specific spells, for example Hydra Token
